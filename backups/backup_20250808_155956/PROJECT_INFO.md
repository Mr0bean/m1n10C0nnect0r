## 核心功能
- MinIO 文件管理系统

- 文档处理管道（Document Pipeline）
- Newsletter 文章管理
- 模糊搜索和推荐系统

## API 端点
- /api/v1/buckets/* - 存储桶管理
- /api/v1/objects/* - 文件对象管理
- /api/v1/search/* - 搜索功能
- /api/v1/documents/* - 文档搜索和推荐
- /api/v1/newsletter/* - Newsletter 管理
